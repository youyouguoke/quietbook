This quiet book takes children on a playful journey through the prehistoric world. Each page invites little hands to match dinosaurs, build a skeleton, sort herbivores from carnivores and count ancient creatures.

## What's Included

- Dinosaur Matching activity
- Fossil Puzzle page
- Dinosaur Sorting game
- Counting Dinosaurs page
- Build a Skeleton activity
- Cut-out pieces at the end of the book

## Skills Developed

- **Fine Motor Skills:** Cutting, placing and Velcro matching strengthen little fingers.
- **Matching:** Pair dinosaurs with footprints and eggs.
- **Science:** Learn basic dinosaur facts and classification.
- **Problem Solving:** Complete the skeleton puzzle and sorting challenges.

## How to Use

1. Download the PDF and print all pages.
2. Cut out the activity pieces on the last pages.
3. Laminate pages for durability.
4. Attach Velcro dots to make every activity reusable.
